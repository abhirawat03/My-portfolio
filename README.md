# abhishek.rawat() - Developer Portfolio

A responsive, high-performance developer portfolio website built using **Vite + React** and styled with **Sleek Dark Mode**. Structured to showcase backend-heavy and full-stack software engineering capabilities.

## 🚀 Key Features

*   **🖥️ Interactive Terminal Console**: Tabbed terminal demonstrating profile configuration and mock server responses (caching checks, Socket connection logs).
*   **✍️ Dynamic Typewriter Titles**: monospaced title animation cycling through roles with a blinking cursor.
*   **📊 Viewport-Triggered Skill Bars**: Progress indicators that smoothly animate when scrolled into view.
*   **✨ Scroll Entrance Animations**: Subtle fade-in and slide-up effects on cards and text blocks as you browse.
*   **📁 Custom Branded Favicon**: `{ AR }` vector logo optimized for browser tabs and bookmarks.

---

## 🛠️ Technology Stack

*   **Core**: React 19, JavaScript (ES6+)
*   **Build Tool**: Vite 8
*   **Styles**: Custom Vanilla CSS (featuring glassmorphism, responsive grids, variables, and keyframe animations)
*   **Icons**: Lucide React & Native inline SVGs

---

## 💻 Local Setup & Development

### 1. Reinstall Dependencies
If you have just cloned or moved this repository, reinstall packages by running:
```bash
npm install
```

### 2. Start Dev Server
To start the local development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### 3. Production Build
To generate an optimized distribution bundle in the `dist` folder:
```bash
npm run build
```
